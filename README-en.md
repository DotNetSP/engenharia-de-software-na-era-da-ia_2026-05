# engenharia-de-software-na-era-da-ia_2026-05
Photos and general information about the event "Software Engineering in the Age of AI", held in São Paulo, Brazil.

Date: **05/30/2026 (Saturday)**

Organizers:
- **Renato Groffe (Microsoft MVP, Docker Captain, Grafana Champion, APIsec U Ambassador, MTAC)**
- **Fábio Araújo (QA Lead, Dev Referências)**
- **Daniela Pacheco (Vericode)**
- **Maria Fabiana (Vericode)**

Number of attendees: **35 people**

A recording of the presentations is available on **YouTube**: **https://www.youtube.com/watch?v=YotQYkVlDc0**

---

Presentations/talks that took place during the event:

_# MCP + AI + Performance with k6 + Observability_

Speakers: **Fábio Araújo (Dev Referências), Renato Groffe (Microsoft MVP, Docker Captain, Grafana Champion, APIsec U Ambassador MTAC)**

Technologies and topics covered: **Grafana k6, MCP, Visual Studio Code, GitHub Copilot, Observability, Artificial Intelligence, LLMs, MCP, DevOps, Azure Container Apps, OpenTelemetry, Application Insights, Azure Monitor, Docker, ASP.NET Core, .NET, Docker, Containers...**

_# Challenges of the regulated market in the age of AI_

Speaker: **Marcelo Marchi (CEO and Founder of Vericode)**

Technologies and topics covered: **Artificial Intelligence, LLMs, MCP, GitHub Copilot, Claude, Software Development, Web Development, Software Quality...**

_# Panel: AI-supported Governance_

Participants:
- **Maria Fabiana (Vericode)**
- **Marcelo Marchi (CEO and Founder of Vericode)**
- **Juliana Lopes (Microsoft MVP)**
- **Renato Groffe (Microsoft MVP, Docker Captain, Grafana Champion, APIsec U Ambassador MTAC)**

Technologies and topics covered: **Artificial Intelligence, LLMs, MCP, GitHub Copilot, Claude, Software Development, Web Development, Software Quality, Security, OWASP Top 10 LLM, OWASP Top 10 MCP, Cybersecurity...**
